# nodoso2social
Publicación de valores de dióxido de azufre proveniente de sensor en GNUsocial

```
git clone https://github.com/gurumelo/nodoso2social
cd nodoso2social
npm install
nano confi.json
node index.js
```

El archivo pde debe compilarse y cargarse en la placa
